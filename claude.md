## Aesthetic Direction
ANTIGRAVITY — zero-gravity cinematic UI
- Objects, text blocks, and cards float and drift slowly in 3D space
- Deep space black background with subtle depth fog
- Elements tilt on mouse move (parallax tilt, no gravity anchor)
- Scroll = moving through space, not down a page
- Floating UI panels like astronaut HUD displays
- Typography: large, weightless — letters feel like they could drift off
- Color: deep black #0a0a0f, electric white #f0f0ff, cold blue accent #4fc3f7
- Particle field background — dust/debris floating in zero-G (Three.js)
- No hard borders — elements bleed into space
- Subtle rotation on all floating cards (2-3 deg drift loop)

## Tech Stack
- Pure HTML + CSS + Vanilla JS (GitHub Pages compatible)
- Three.js — floating particle debris field
- GSAP — floating drift animations, scroll-through-space transitions
- VanillaTilt.js — mouse-tracked 3D tilt on all cards
- Google Fonts: Space Grotesk + JetBrains Mono

## Scroll Story (antigravity narrative)
1. HERO — Name drifts in from deep space, letter by letter, no gravity

4. SKILLS — Floating orbs/chips, each skill a drifting object
